Interface CLI e GUI

CLI

CLI é a sigla de Command-Line Interface, que traduzido significa “Interface de Linha de Comando”.
As CLIs são uma forma de interação do usuário com o computador, que é feita através de comandos de
texto. O que possibilita a execução de várias tarefas.

GUI

a sigla GUI refere-se a denominação "Graphical User Interface" (Interface Gráfica do Usuário, em português),
que consiste em um modelo de interface do utilizador que permite a interação com os dispositivos digitais através de elementos gráficos.
